# Burn Rate Monitor 概要设计

## 1. 目录结构

```
burn-rate-monitor/
├── src/
│   └── brmonitor/
│       ├── __init__.py
│       ├── data_loader.py
│       ├── calculator.py
│       ├── server.py
│       └── public/
│           └── index.html
├── docs/
├── pyproject.toml
├── CLAUDE.md
└── README.md
```

## 2. 模块概要

### 2.1 data_loader模块

- 扫描Claude数据目录
- 解析JSONL文件
- 返回使用记录列表

### 2.2 calculator模块

- 按分钟聚合数据
- 计算burn rate
- 支持多种时间范围

### 2.3 server模块

- FastAPI应用
- 提供REST API
- 托管静态文件

### 2.4 前端页面

- 深色主题
- ECharts图表
- 统计卡片

## 3. API设计

| 端点 | 方法 | 描述 |
|------|------|------|
| `/` | GET | 前端页面 |
| `/api/burn-rate` | GET | burn rate数据 |
| `/api/stats` | GET | 统计摘要 |

## 4. 配置

- 端口: 3001 (默认)
- 数据目录: 自动检测
